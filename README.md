# How to modify using Git?

Clone the git repository
```sh
 `git clone https://github.com/maverickw5/online-judge-platform.git`
 ```

1. Pull
```sh
 `git pull`
 ```
2. Add changes
For all modified files
```sh
 `git add .`
 ```
For targeted file
```sh
 `git add {filename}`
 ```
3. Commit
```sh
 `git commit -m '{message}'`
 ```
3. Push
```sh
 `git push`
 ```
